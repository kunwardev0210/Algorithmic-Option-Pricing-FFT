# Algorithmic Option Pricing Engine using Fourier Transform

A high-performance quantitative financial engine developed to calculate option prices and analyze market arbitrage dynamics using mathematical models, stochastic calculus, and the Fast Fourier Transform (FFT) algorithm.

This project was built under **Stamatics, IIT Kanpur**.

---

## 📌 Project Overview

Traditional numerical methods for option pricing can be computationally expensive when dealing with complex financial derivatives. This project implements an algorithmic option pricing engine in Python, leveraging advanced stochastic calculus tools and accelerating calculation throughput **10x** by integrating a high-performance FFT-based approach (such as the Carr-Madan methodology).

---

## 🚀 Key Features & Methodology

### 1. Financial Modeling & Derivative Analysis
* **Core Foundational Models:** Implemented foundational quantitative finance frameworks to compute accurate derivative evaluations.
* **Complex Derivatives:** Evaluated **4 complex financial derivatives** while modeling real-world market arbitrage dynamics.
* **Probability Foundations:** Utilized robust probability theory concepts to map asset distribution profiles.

### 2. Stochastic Calculus Foundations
* Mastered and applied advanced **Stochastic Calculus** tools to construct continuous-time financial models:
  * **Standard Brownian Motion Models** (Wiener processes) to simulate stochastic asset paths.
  * **Ito's Lemma** to determine the differentials of time-dependent functions of stochastic processes.

### 3. FFT-Driven Optimization
* **10x Computational Speedup:** Achieved massive performance acceleration for complex option pricing by implementing a high-performance **Fast Fourier Transform (FFT)** algorithm.
* Replaced slower, iterative numerical integration methods with characteristic-function-based Fourier inversions.
